## 2026-05-12 - LLMAdjudicator Unexpected JSON Type **Edge Case:** Valid JSON that parses to a list instead of a dict returned by the LLM provider. **Learning:** Internal methods fail with AttributeError, but outer evaluate() safely catches the generic exception and returns 0.0. **Action:** Always test parsed JSON responses for expected types (e.g., dict vs list vs boolean) before calling dictionary-specific methods.
