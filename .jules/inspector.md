## 2024-05-13 - LLMAdjudicator returns 0.0 on malformed list JSON response **Edge Case:** LLMProvider returns valid JSON that parses to a list or boolean instead of a dictionary. **Learning:** Internal dictionary unpacking methods fail with AttributeError or TypeError, but `evaluate` catches the generic exception and safely returns 0.0. **Action:** Added test `test_llm_adjudicator_malformed_json_type` to cover when valid JSON isn't a dictionary.
