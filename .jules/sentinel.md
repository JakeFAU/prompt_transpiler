## 2024-05-06 - [Dynamic SQL Column Injection] **Vulnerability:** [SQL injection in `update_job` due to unsanitized dictionary keys used as column names] **Learning:** [Database drivers cannot parameterize column names, only values. Untrusted dictionary keys passed to UPDATE queries via string interpolation allow arbitrary SQL execution] **Prevention:** [Always validate dynamically generated column names against an explicit allowlist or use `.isidentifier()` before string interpolation]
