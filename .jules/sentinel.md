## 2026-04-29 - Fix dynamic SQL column injection **Vulnerability:** SQL injection vulnerability via dynamically constructed UPDATE queries using f-strings with unvalidated column names. **Learning:** Dynamic string interpolation (f-strings) in SQL queries is unsafe, even for column names, as parameterized queries only work for values. **Prevention:** Always validate dynamically provided column names using `isidentifier()` to prevent injection, then suppress false-positive static analysis alerts with .
