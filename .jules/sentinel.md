## 2026-04-26 - [Job Store DB SQL Injection via dictionary keys] **Vulnerability:** [SQL injection vector in _to_update_clause using unparameterized string formatting for column names from user-supplied dictionaries] **Learning:** [Dictionary keys are not parameterized in SQLite/DuckDB parameter substitutions, leading to SQLi if directly interpolated] **Prevention:** [Use .isidentifier() to strictly validate keys before interpolating into SQL queries]
