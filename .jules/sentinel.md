## 2024-05-18 - [Fix SQL Injection Vector] **Vulnerability:** Unparameterized column names in SQL UPDATE statement. **Learning:** Dynamic dictionaries passed as **kwargs can be used to inject unsafe column names when directly f-string interpolated into SQL statements. Standard parameterization (?) only works for values, not identifiers. **Prevention:** Use a strict allowlist (e.g., frozenset of valid field names) to validate dictionary keys before building dynamic query strings.
