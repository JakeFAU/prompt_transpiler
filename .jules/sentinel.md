## 2026-05-02 - SQL Injection in dynamic UPDATE queries **Vulnerability:** Potential SQL injection via dynamic column names in f-string query construction in _to_update_clause. **Learning:** Standard parameterization cannot be used for SQL identifiers like column names, so string formatting was used instead, creating a blind spot for injection. **Prevention:** Always use .isidentifier() on dynamically constructed or user-provided column names before string interpolation in SQL queries.
