## 2026-05-08 - Prevent SQL Injection via hardcoded string interpolation **Vulnerability:** Unsanitized string interpolation used for SQL query in DuckDBJobStore and SQLiteJobStore update methods **Learning:** Dynamic column generation used string interpolation (f-strings) instead of parameterization for updating rows. **Prevention:** Always use parameterized query structures for values, and when columns are dynamically generated validate they are valid column identifiers to prevent injection risks
