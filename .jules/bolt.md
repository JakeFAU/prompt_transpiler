## 2025-02-20 - [Enum Attribute Lookup Overhead in Inline Sets] **Learning:** [Sets constructed dynamically with Enum attributes (like `{JobStatus.SUCCEEDED.value}`) are NOT compiled into `frozenset` at compile-time by CPython. This causes O(n) set instantiation and attribute lookups inside hot loops and comprehensions.] **Action:** [Always define sets that contain object attribute lookups or dynamically computed values as module-level `frozenset` constants when used in hot loops.]
