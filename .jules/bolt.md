## 2025-04-24 - [Optimize in-memory job store purge via constant set] **Learning:** In Python, sets containing attribute lookups (e.g., Enum values) are not optimized into `frozenset` at compile time. Defining them dynamically inside loops or comprehensions creates an O(n) creation overhead per iteration. **Action:** Define these sets as module-level `frozenset` constants to ensure O(1) membership testing and avoid repeated instantiation overhead.
